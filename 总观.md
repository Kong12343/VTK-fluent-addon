# 端到端 AI 驱动的非结构网格生成与流场预测框架 (v2.0)

## 1\. 框架总览 (Framework Overview)

本项目旨在构建一个完全由数据驱动的“神经 CFD 求解器”。针对复杂三维进气道（S弯、Y型分叉等）的设计与优化，通过\*\*“分而治之”\*\*的监督学习策略，实现从“带多维物理标签的边界面”直接端到端输出“含流场数值的完整内部计算图”。

**核心三大模块：**

1.  **体积折叠网络 (Volumetric FoldingNet):** 负责宏观坐标生成（点云回归），白嫖结构化基础拓扑。
2.  **神经连边器 (Neural Edge Predictor):** 负责微观拓扑修剪（二分类），解决物理边界穿透与网格畸变。
3.  **流场 GNN (MeshGraphNets):** 负责在动态生成的图结构上进行物理信息传播（场回归）。

## 4\. 数据特征工程与多维边界对齐 (Data Features & Alignment)

流体物理量和几何尺度的跨度极大，且不同边界类型的物理量维度不同。本框架采用 **“补零门控 + 独立归一化”** 机制解决多维特征输入问题。

### 4.1 多维特征张量构造表

| 节点类型 | 变量名 | 维度 | 物理特征描述 |
| :--- | :--- | :--- | :--- |
| **Boundary Face** | $X_{geo}$ | 3 | 边界型心坐标 $(x,y,z)$ |
| | $X_{norm}$ | 3 | 边界面法向向量 $(n_x, n_y, n_z)$ |
| | $X_{label}$ | $C$ | 边界类型独热码，如 `[1,0,0]`(Wall), `[0,1,0]`(Inlet) |
| | $X_{val}$ | $K$ | **统一扩充为 K 维的多物理量张量** (详见 4.2) |
| **Internal Cell** | $X_{pos}$ | 3 | 内部控制体质心坐标 $(x,y,z)$ |
| **Topology Edge** | $E_{attr}$ | 4 | 空间相对位移 $(dx, dy, dz)$ + 欧氏距离 $||d||$ |
| **Target Field** | $Y_{target}$ | $D_{out}$ | 待预测的物理场真值 (如速度、压力) |

### 4.2 异构边界的对齐与门控机制 (Padding & Label Gating)

针对不同边界包含不同物理量（如 Wall 无速度压力输入，Inlet 有速度无压力），执行以下策略：

1.  **统一维度 (Zero-Padding):** 将所有边界的物理量强行拼接为固定长度 $K$ 的向量（例如 `[u, v, w, P, T, k, omega]`）。对于该边界缺失的物理量，直接用 `0.0` 填充。
2.  **标签门控 (Label Gating):** 神经网络利用输入的独热码 $X_{label}$，通过注意力机制自动学习关闭（Mask）全零无效通道，避免补零对特征提取造成干扰。

### 4.3 独立通道归一化 (Independent Channel Z-Score)

各物理量必须**分别独立**进行标准化。计算均值和方差时，**必须利用 Mask 排除掉补零的假数据**，仅统计有效数值。
$$\hat{X}_{val}^{(k)} = \frac{X_{val}^{(k)} - \mu^{(k)}}{\sigma^{(k)}} \quad \text{for } k \in \{1, 2, \dots, K\}$$

## 5\. 详细架构实现 (Implementation Details)

### 阶段一：C++ 数据零拷贝提取

在 `vtkFLUENTCFFReader` 中，解析网格并将各类边界数据提取并组装为等长的 $K$ 维 Tensor，通过 `torch::from_blob` 零拷贝传回 Python。

### 阶段二：宏观骨架生成 —— 三维体积折叠网络 (3D Volumetric FoldingNet)

1.  使用 PointNet++ 对包含多维边界条件的总输入 `[N, 3+3+C+K]` 进行编码，提取全局隐向量 $Z$。
2.  初始化 3D 晶格坐标 $Grid_{3D}$，其自带标准的六面体拓扑。
3.  通过形变场解码器：$X_{internal} = MLP(Grid_{3D} \oplus Z)$，白嫖基础网格拓扑。

### 阶段三：微观拓扑修补 —— 神经连边器 (Neural Edge Predictor)

1.  使用 `torch_cluster.radius_graph` 在大曲率和附面层区域生成局部候选边。
2.  提取边特征 $E_{ij} = [X_i, X_j, X_i-X_j, Z]$。
3.  通过 Edge MLP 结合 Gumbel-Softmax 输出边的保留概率 $P_{ij}$，惩罚物理穿透与过度畸变。

### 阶段四：物理场求解 —— 异构流场 GNN (MeshGraphNets)

基于生成的坐标与合法拓扑，执行 10-15 层残差空间异构图卷积，完成动量与压力的全场扩散求解。

### 阶段五：联合训练损失函数 (Joint Optimization)

$$\mathcal{L}_{total} = \alpha \mathcal{L}_{Chamfer}(点云坐标) + \beta \mathcal{L}_{BCE}(拓扑连边) + \gamma \mathcal{L}_{MSE}(多维流场)$$
